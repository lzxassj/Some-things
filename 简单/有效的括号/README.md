## 整数反转

给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串，判断字符串是否有效。

有效字符串需满足：

1. 左括号必须用相同类型的右括号闭合。
2. 左括号必须以正确的顺序闭合。
3. 注意空字符串可被认为是有效字符串。


**示例1：**

```javascript
输入: "()"
输出: true
```



**示例2:**

```javascript
输入: "()[]{}"
输出: true
```



**示例3:**

```javascript
输入: "(]"
输出: false
```



**示例4:**

```javascript
输入: "([)]"
输出: false
```



**示例5:**

```javascript
输入: "{[]}"
输出: true
```
