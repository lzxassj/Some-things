## 相同的树
<font color=#999999>标签：树，递归，迭代</font>



给定一个二叉树，检查它是否是镜像对称的。

例如，二叉树 [1,2,2,3,4,4,3] 是对称的。
```javascript
    1
   / \
  2   2
 / \ / \
3  4 4  3
```

